### 🏆 Featured Projects

#### ⭐ [ContextL](https://github.com/DS0710-coder/ContextL) — *My Flagship Project*
> **Repository Intelligence Engine that helps AI understand codebases beyond vector embeddings.**

<table>
<tr>
<td width="65%">

While traditional AI tools rely solely on flat vector embeddings, **ContextL** models entire repositories as **structured knowledge graphs** and **Abstract Syntax Trees (ASTs)**. It maps relationships between files, classes, and symbols, enabling LLMs to navigate and reason about complex codebases with surgical precision.

* **Key Tech:** `Tree-sitter` • `NetworkX` • `BM25 Sparse Retrieval` • `MCP Server Integration` • `Python`
* **Why it matters:** Brings structural graph logic to AI coding assistants for deeper codebase comprehension.
